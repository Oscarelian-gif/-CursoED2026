## Descripción

Programa desarrollado en lenguaje C para gestionar el peso de hasta 30 contenedores en un patio de carga.

El sistema permite:

* Registrar pesos manualmente.
* Generar pesos aleatorios.
* Mostrar los contenedores registrados.
* Ordenar los contenedores mediante métodos de ordenamiento eficientes.
* Realizar búsquedas binarias sobre un arreglo ordenado.
